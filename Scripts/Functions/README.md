# Future battery recycling in North America: Capacity expansion under shifting demand and circularity strategies

Define common user-built functions used throughout the model.

* **StockFlows_EV_LIB.R**: Function to get stocks and outflows of EVs and LIBs using product-component failure method.
* **Survival_EV_LIB.R**: Survival Curve based on Logistic Distribution. Function to get flows (numbers of cars,EV,LIB) depending on the vehicle and battery starting age.
